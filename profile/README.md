# 语雀 Yuque

> 笔记与文档知识库工具，面向企业、组织或个人。

**语雀**是一款优秀的知识管理工具，帮助个人和团队沉淀、组织、分享知识。

### Quick Start

```bash
# Use with Claude Code
claude mcp add yuque-mcp -- npx -y yuque-mcp --token=YOUR_TOKEN

# Use with any MCP-compatible client
npx -y yuque-mcp --token=YOUR_TOKEN
```
